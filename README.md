# React App Mini Social Network

### Build with
- React
- Redux
- Html
- Css
- Axios

### Features
- Login/Logout account;
- Change profile photo
- Edit personal info
- Follow, unfollow other users
- Change status

### Installing 
To install project, please download or clone code and after use:
```sh
npm i

npm start  
```
